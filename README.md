
# Building Login & Signup page for React Application




## Run Locally

Clone the project

```bash
  git clone https://github.com/anishashinde2507/Login_Signup.git
```

Go to the project directory

```bash
  cd client
```


Start the server

```bash
  npm run dev
```
Backend Installation


```bash
  cd server
```

```bash
  npm i express cors mongoose nodemon
```


```bash
  nodemon index.js
```
